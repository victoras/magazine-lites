== Description ==
Magazine Lites is a dynamic and visually stunning Free WordPress Theme specially designed for Magazines, Blogs, Fashion & Lifestyle Blogs. The theme is easy to set up and customize, you need just a few clicks, and you can entirely change the look of the website!

== Easy to install ==
The installation process is quite simple and easy, it takes just a few minutes!

Activate the theme and go to `Appearance > Customizer`.

## Toolbar
All options from this section will appear to the toolbar

## Social
Add social Icons which will be used for toolbar and header area

## Other
Other option related to post Layout

## Footer Options
Option to add the Copyright

## The theme has 2 custom templates:
1. Home page template - To display Home page
3. Full Width - to display full width page
